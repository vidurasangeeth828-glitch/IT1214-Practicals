====Project Description====
This is a simple Warehouse Inventory Management System built using Java and ArrayList.  
The program allows users to:
      - Add new items to inventory
      - Remove existing items
      - Update item quantity
      - Search items by ID or name
      - Display all items
====How to Run the Program====
1. Open cmd and navigate to the project folder.
2. compile Werehouse.java file
3. run the Werehouse(java Warehouse).
